# familiar95
This is a hobby project of recreating a Windows 95-like desktop in the browser using Vue.js.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
